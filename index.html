    Training

    Diligent offers various Industrial Training programmes. Scroll down to know more.

    AllenBradley - Controllogix & Compactlogix

    This course is aimed for Allen-Bradley PLC users (Logix Platform) who want to maintain or program a Logix5000 project.

    • RS Logix Studio 5000 Overview / RS Linx
    • CPU Modes / Configuring Ethernet Modules
    • Downloading / Uploading
    • Tag Database
    More...

    AllenBradley - Controllogix & Compactlogix (Advanced)

    This course is aimed for Allen-Bradley PLC users (Logix Platform) who want to maintain or program a Logix5000 project.

    • Enhanced PID Instruction
    • Function Block Boolean Logic / Editing
    • Structured Text Basics / ST Conditionals / Case Statements / Loops
    • SFC Editing / Steps and Transitions / Actions / Forcing
    More...

    Industrial Networks

    This course provides you with the skills necessary to efficiently design and configure an Industrial network.

    • Introduction to Industrial data networks
    • Serial connections; RS232/RS422/RS485 and Protcols; DF1 / DH-485 / Modbus / ASCII
    • Upgradation or Enhancement of Automation Systems
    • ModbusTCP / Profibus / Devicenet / ControlNet / Ethernet
    More...

    AllenBradley - Kinetix CIP Servo

    This course is designed to provide candidates with an understanding of the concepts, terminology, functionality and applications of motion control.

    • Identify Servo motion elements / Motion drive elements
    • Configuring a Studio 5000 Logix Designer Project for Integrated Motion over an EtherNet/IP Network
    More...

    AllenBradley - Micrologix & SLC

    This course is aimed for Allen-Bradley PLC users ( Micrologix / SLC) who want to build up a weighty knowledge about PLC (SLC & Micrologix Platform) and programming software.

    • Overview of the Micrologix & SLC-500 components
    • RS Logix 500 software package
    • Shift Register Instructions / FBC and DDT Instructions
    • The PID Instruction
    More...

    AllenBradley - SCADA Factorytalk ViewSE

    This course is intended for plant engineers, control specialists, technicians and other individuals who need to understand the fundamental concepts of creating highly functional HMI applications for local or distributed processes.

    • Getting Started with Factory Talk View SE Software
    • Creating and Configuring a Local Factory Talk View SE Application
    • Creating a Graphic Display and Adding Basic Objects in a Factory Talk
    More...
